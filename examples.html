Examples

Case insensitive

{% highlight javascript %} $('textarea').highlightTextarea({ words: ['lorem ipsum', 'VULPUTATE'], caseSensitive: false }); {% endhighlight %}

Resizable

{% highlight javascript %} $('textarea').highlightTextarea({ words: ['Lorem ipsum', 'vulputate'], resizable: true }); {% endhighlight %}

Regular Expression

{% highlight javascript %} $('textarea').highlightTextarea({ words: ['{(.+?)}'] }); {% endhighlight %}

Multiple colors

{% highlight javascript %} $('textarea').highlightTextarea({ words: [{ color: '#ADF0FF', words: ['Lorem ipsum', 'vulputate'] }, { color: '#FFFF00', words: ['Donec'] }] }); {% endhighlight %}

Indexes ranges

{% highlight javascript %} $('textarea').highlightTextarea({ ranges: [{ color: '#ADF0FF', start: 0, length: 10 }, { color: '#FFFF00', ranges: [[40,45], [50,55]] }] }); {% endhighlight %}
